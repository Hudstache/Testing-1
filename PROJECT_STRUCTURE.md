# Archipelago Adventure Project Structure

## Main pages

### index.html
The game's main menu.

### game.html
The playable island screen.

## CSS

### css/menu.css
Controls the appearance of the main menu.

### css/game.css
Controls the appearance of the game screen and mobile controls.

## JavaScript

### js/menu.js
Controls main-menu buttons, settings, save-slot selection and navigation.

### js/game.js
Controls the game world, player movement, interactions, inventory and resource gathering.

## Assets

### assets/sprites
Character, object, island and animation images.

### assets/music
Background music files.

### assets/sounds
Sound effects such as footsteps, chopping and collecting resources.

### assets/fonts
Local game fonts.

## Maps

### maps
Island layouts and world data.
